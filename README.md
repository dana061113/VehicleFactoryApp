# Vehicle Factory Method (Java)

Бұл жоба **Factory Method** жобалау паттернін пайдалана отырып,
әртүрлі транспорт құралдарын (көлік түрлерін) құруға арналған
кеңейтілетін жүйені көрсетеді.

Жоба **Java тілінде** жазылған және университеттегі
«Шаблоны проектирования приложений» пәніне арналған.

---

## 🎯 Мақсаты

- Factory Method паттернін практикада қолдану
- Әртүрлі транспорт түрлерін бірдей интерфейс арқылы басқару
- Жүйені жаңа транспорт түрлерімен оңай кеңейту

---

## 🧩 Қолданылған паттерн

**Factory Method**

Паттерннің мәні:
- Клиент код нақты классты (`Car`, `Bus`, т.б.) білмейді
- Объектілер фабрика арқылы құрылады
- Жаңа транспорт қосқанда ескі код өзгермейді (OCP принципі)

---

## 🚗 Қолдау табатын транспорттар

- Car (Автомобиль)
- Motorcycle (Мотоцикл)
- Truck (Жүк көлігі)
- Bus (Автобус)

Әр транспорт `IVehicle` интерфейсін жүзеге асырады.
